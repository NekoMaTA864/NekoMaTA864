##  My Projects

###  [Discord 自動化圖文管家 (Telegram → Discord)]([這裡是連結](https://github.com/NekoMaTA864/tg-to-discord-bot))

- **Tech Stack:** `Python` `Discord API` `Telegram API`
- **技術重點：** API 串接、訊息轉發流程設計、簡單分類邏輯
- **為什麼做：** 平時追蹤太多頻道，同時開著 Telegram 和 Discord 很麻煩，
  找存過的內容也不方便，因此寫了一個橋接機器人，
  將圖文訊息自動分類並同步到私人的 Discord，現在只需要集中在一個地方管理。
- **進度：**  穩定運作中（第一階段完成），下一步將優化大檔案傳輸能力。

###  [基於 OpenCV 的遊戲自動化小助手](這裡是連結)

- **Tech Stack:** `Python` `OpenCV` `ADB`
- **技術重點：** 畫面辨識（Template Matching）、操作模擬、自動化流程控制

  應用在實際情境中，減少重複操作的時間成本。
- **進度：**  核心功能完成，目前可穩定辨識畫面並執行操作，
  下一步將優化更複雜的判斷邏輯。
